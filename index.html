Tecnologias

HTML

"Tenho uma boa noção das tags HTML principais e sou capaz de criar estruturas mais complexas como: listas, formulários e imagens. Embora eu consiga formatar textos e adicionar mídias, ainda estou aprendendo sobre boas práticas de semântica e acessibilidade."

CSS

"Tenho um bom entendimento das propriedades CSS mais comuns e consigo estilizar páginas com layouts simples. Me sinto confortável com CSS e posso criar layouts responsivos usando Flexbox e Grid. Sei aplicar estilos com eficiência, utilizando de algumas pseudo-classes e pseudo-elementos, sei também estilizar uma página utilizando um pouquinho de animações, e também consigo adaptar minha páginas para diferentes dispositivos através do uso de media queries. Contudo, ainda tenho dificuldade com designs mais complexos e busco melhorar minha consistência em projetos maiores."

Javascript

"Tenho familiaridade com a sintaxe do JavaScript e consigo realizar operações e interações básicas em páginas web. Consigo manipular elementos simples, mas ainda tenho muito a aprender sobre a linguagem e suas funcionalidades."
